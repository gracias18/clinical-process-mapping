# Clinical Process Mapping

## Overview
End-to-end clinical workflow mapping using BPMN 2.0 notation to identify 
handoff inefficiencies and digitization opportunities.

## What Was Mapped
Workflow of a patient and her caregiver, physician, and clerk at a family medical clinic.

## Methodology
- Modelled processes using BPMN 2.0 notation
- Identified inefficiencies, redundancies, and handoff gaps

## Tools
- BPMN 2.0
- Camunda

## Output
See `MHI2017 BPMN Markup.pdf` for the full workflow diagram.
